# Novalights

A modern, high-end e-commerce website for professional workshop lighting solutions.

## Features
- **Sleek Design**: Modern, white-themed aesthetic with glassmorphism effects.
- **Product Gallery**: Showcase of real-world installations.
- **Detailed Product Page**: Professional product listing with configuration options.
- **Responsive**: Fully optimized for mobile and desktop.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser to view the home page.
3. Open `product.html` to view the product details.

## Technologies
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
